# CarDVR — Android-приложение для видеорегистратора

Управление, просмотр в реальном времени и воспроизведение записей с WiFi-видеорегистраторов на базе протокола **uCam / SiCaDroid** (совместимо с uCarDvr).

Оптимизировано под **Poco X6 Pro** (6.67", 1220×2712, 144 Гц).

---

## Функции

| Модуль | Описание |
|---|---|
| **Подключение** | WiFi-подключение к регистратору по IP/порту с сохранением профиля |
| **Live Preview** | Просмотр видео в реальном времени по RTSP, управление записью |
| **Запись** | Старт/стоп, фото, блокировка файла от перезаписи |
| **Список файлов** | Видео и фото с SD-карты, удаление, блокировка, форматирование |
| **Воспроизведение** | Полноэкранный плеер: прогресс, скорость ×0.5…×4, ±10 сек |
| **Настройки — Общие** | Разрешение (1080p/720p/480p), петля, экспозиция, баланс белого, звук, штамп |
| **Настройки — Безопасность** | Режим парковки, детекция движения, чувствительность G-сенсора |
| **Настройки — ADAS** | Выезд за полосу (LDWS), дистанция (FCWS), предупреждение о скорости |
| **Настройки — WiFi** | SSID, пароль, IP, порт |
| **Настройки — Режим сна** | Автовыключение, задержка, охрана парковки |

## Сборка

### Требования
- Android Studio Hedgehog или новее
- JDK 17
- Android SDK 35

### Шаги
```bash
git clone https://github.com/<username>/CarDVR
cd CarDVR
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`

### CI/CD (GitHub Actions)
Push в `main` или `develop` автоматически собирает APK.  
Release-сборка с подписью требует секретов в репозитории:
- `KEYSTORE_BASE64` — keystore в base64
- `KEY_ALIAS`, `KEY_PASSWORD`, `STORE_PASSWORD`

## Подключение к регистратору

1. Включите регистратор
2. На телефоне перейдите в WiFi → подключитесь к сети регистратора (обычно `UCAM_DVR` / пароль `12345678`)
3. Откройте CarDVR → введите IP `192.168.1.1`, порт `7878`
4. Нажмите **Подключиться**

---

## Протокол

Приложение использует **TCP JSON-команды** поверх постоянного сокета (порт 7878):

| msg_id | Команда |
|---|---|
| 257 | START_SESSION |
| 513 | START_RECORD |
| 514 | STOP_RECORD |
| 515 | TAKE_PHOTO |
| 769 | GET_FILE_LIST |
| 770 | DELETE_FILE |
| 771 | LOCK_FILE |
| 1025 | FORMAT_SD |

Видеопоток — **RTSP** на порту 554: `rtsp://192.168.1.1:554/stream0`

## Архитектура

```
Clean Architecture + MVVM
├── data/
│   ├── model/       — модели данных
│   ├── network/     — DvrTcpClient (TCP-протокол)
│   └── repository/  — DvrRepository (DataStore + сеть)
├── di/              — Hilt DI
├── service/         — Foreground Service
└── ui/
    ├── components/  — переиспользуемые компоненты
    ├── screens/     — экраны (Compose)
    ├── theme/       — цвета, типографика
    └── MainViewModel.kt
```

**Стек:** Kotlin · Jetpack Compose · Hilt · ExoPlayer (Media3) · Retrofit · Room · DataStore · Coil · Navigation Compose
